O que é Apache?

O Apache é um servidor de código aberto e nome oficial é Apache HTTP Server, mantido pela Apache Software Foundation, e alimenta cerca de 46% de todos os sites hospedados na internet. O Apache permite que donos de sites mostrem e mantenham seus conteúdos na internet e daí vem o nome de “servidor de internet”.
